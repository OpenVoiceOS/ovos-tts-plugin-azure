### ovos-tts-plugin-edge-tts

Configuration parameters:

```json
"tts": {
    "module": "ovos-tts-plugin-edge-tts",
    "ovos-tts-plugin-edge-tts": {
        "voice": "nl-NL-MaartenNeural",
        "rate": "+200%",
        "output_file": "edge_tts_output.wav"
    }
}
```

##### Installation

`pip install ovos-tts-plugin-edge-tts`
